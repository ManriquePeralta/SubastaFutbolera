# Subasta Futbolera

Aplicación web para gestionar subastas de jugadores de fútbol con sistema de formaciones y plazas.

## Características

- Subasta por plazas según formación elegida
- Sistema de "paso" único por plaza
- Visualización de planteles a mitad y final de subasta
- Interfaz web interactiva
- Notificaciones en tiempo real

## Tecnologías

- Python/Flask
- JavaScript
- HTML/CSS

## Instalación Local

```bash
# Instalar dependencias
pip install -r requirements.txt

# Ejecutar servidor
python app.py
```

La aplicación estará disponible en `http://127.0.0.1:5000`
